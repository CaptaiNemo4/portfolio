# Portfolio - Chorobek Sheranov

Data Science portfolio website built with Quarto and Shiny for Python.

## Live Site

https://CaptaiNemo4.github.io/portfolio/

## Projects

1. **Bayesian Volatility Modeling** - GARCH(1,1) vs Stochastic Volatility on S&P 500
2. **Portfolio Optimization** - Efficient Frontier, Max Sharpe, GMV, Risk Parity

## Running the Shiny App Locally

```bash
pip install -r requirements.txt
cd projects/data
shiny run app.py
```

## Building the Site

```bash
quarto render
```
